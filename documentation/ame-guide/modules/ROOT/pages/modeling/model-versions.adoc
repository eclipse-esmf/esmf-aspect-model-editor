[[model-versions]]
= Create a new version of an Aspect Model

To create a new version of an Aspect Model without deleting the previous one, proceed as follows:

1. Click *Settings*.
+
→ The Settings dialog opens.
2. Navigate to the tab *Namespaces*.
3. Enter the new version of the namespace in the field *Version*.
+
image::load-and-save-models-settings-namespaces-version.png[width=100%]
4. Click *Save*.
5. Click *Save* to confirm your changes.
6. Click *Save* in the xref:getting-started/ui-overview.adoc#toolbar[toolbar].
7. Click *Save to Workspace* in the dropdown.

→ Now a new version of your currently opened model is saved and added to your Workspace.

For more information on namespaces, see xref:namespace/namespaces-references.adoc#namespaces-external-references[Namespaces and external references].
