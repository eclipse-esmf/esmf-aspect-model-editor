Start modeling

info

An existing Aspect Model will be replaced with the new one. Any unsaved changes will be lost.

To start modeling you have three options:

  • start from a new model (see initial ttl code in the text area below)
  • copy paste ttl code of an existing model into the text area below
  • load ttl code from a file by choosing "Load from File" below.

As an expert you may also modify the shown ttl code manually.
